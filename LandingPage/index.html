Propiedad Explicación Breve
font-variant-numeric Permite darle estilo a los números de un texto.
letter-spacing Permite modificar el espacio entre los caracteres de un texto.
text-align-last Controla la alineación del último renglón del texto de varias líneas.

line-height Define la altura de línea para un elemento de texto.

font-variant-numeric

La propiedad "font-variant-numeric" se utiliza para personalizar la apariencia de los números en el texto.

Por ejemplo, se indica que se establezca el diseño de los números a números antiguos en lugar de cifras proporcionales.

Ejemplo:

            
                p.ejemplo1Ej02 {
                    margin-left: 120px;
                    color: purple;
                    font-variant-numeric: oldstyle-nums;
                }

Resultado:

1234567890


letter-spacing

La propiedad "letter-spacing" define la cantidad de espacio entre los caracteres de un texto. Se utiliza con unidades 'em' o 'ex'.

Por ejemplo, se muestra un texto con mayor espacio entre letras.

Ejemplo:

            
                p.ejemplo2Ej02 {
                    margin-left: 120px;
                    color: purple;
                    letter-spacing: 0.5em;
                }

Resultado:

Espaciado entre letras aumentado


text-align-last

La propiedad "text-align-last" establece la alineación del último renglón del texto de un párrafo con muchas líneas. Se le puede aplicar "left", "right" o "center".

Por ejemplo, el párrafo tiene varias líneas y hacemos que se justifique el último renglón, para crear un efecto estéticamente agradable.

Ejemplo:

            
                p.ejemplo3Ej02 {
                    margin-left: 120px;
                    color: purple;
                    text-align-last: center;
                }

Lorem ipsum dolor sit amet, consectetur adipiscing elit. In metus mauris, blandit vitae auctor id, rhoncus id eros. Nullam sit amet nulla ac sapien eleifend ultrices. Curabitur ac dictum metus. Sed augue risus, tempor in justo non, fermentum faucibus nibh.


word-spacing

La propiedad "word-spacing" ajusta el espacio entre las palabras de un texto. Se puede especificar un valor fijo, relativo o incluso negativo.

Por ejemplo, se muestra un espacio adicional de 5px entre cada palabra en el párrafo.

Ejemplo:

            
                p.ejemplo4Ej02 {
                    margin-left: 120px;
                    color: purple;
                    word-spacing: 15px;
                }

Resultado:

Este es un ejemplo de texto con espacio adicional entre las palabras.


line-height

La propiedad "line-height" controla la altura de línea, que es la altura total de una línea de texto, incluyendo el espacio entre líneas y el espacio adicional (si lo hay) que rodea el texto.

Por ejemplo, se establece la altura de la línea a 1.5 veces la altura normal del texto. Esto afectará el espacio entre líneas y el espacio vertical total ocupado por el texto.

Ejemplo:

            
                p.ejemplo5Ej02 {
                    margin-left: 120px;
                    color: purple;
                    line-height: 1.5;
                }

Resultado:

Este es un párrafo de texto normal.

Este es un párrafo de texto con line-height ajustado.

ODS 8: Trabajo Decente y Crecimiento Económico

El Objetivo de Desarrollo Sostenible 8 busca promover el crecimiento económico sostenido, inclusivo y sostenible, el empleo pleno y productivo y el trabajo decente para todos.

Una medida para contribuir al ODS 8 es fomentar la creación de empleo sostenible, proporcionar empleo digno.

El sector privado tiene un rol claro en la consecución de este Objetivo, ya que es un actor principal para el crecimiento económico y la creación de puestos de trabajo.
